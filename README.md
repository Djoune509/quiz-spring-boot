# Quiz Project

## 📖 Description
**Quiz Interactif** est une application web développée avec Spring Boot qui permet aux utilisateurs de tester leurs connaissances en **Mathématiques**, **Anglais** et **Chimie**.  
Les fonctionnalités principales incluent :
- 🎯 **Choix de la matière** (Mathématiques, Anglais ou Chimie).
- 📝 **Réponse aux questions** sans base de données (questions intégrées dans l’application).
- 📊 **Affichage du score final** après avoir terminé le quiz.
- 🔄 **Possibilité de relancer un quiz** pour s’entraîner.

Ce projet vise à offrir une plateforme simple et interactive pour l’apprentissage et l’évaluation.

## 🛠️ Technologies
- Java / Spring Boot
- Render (déploiement)
